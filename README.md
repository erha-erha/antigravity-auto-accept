# Antigravity Auto Accept (修复版)

本插件旨在解决 Antigravity 的命令无法自动运行的问题。

## 功能
此插件会自动接受 Antigravity Agent 的所有待处理步骤，包括“运行命令”和“保存文件”等请求。

## 原项目地址
[https://open-vsx.org/extension/pesosz/antigravity-auto-accept](https://open-vsx.org/extension/pesosz/antigravity-auto-accept)

## 安装方法

1.  **下载插件**：在本仓库的文件列表中找到并下载 `pesosz.antigravity-auto-accept-1.0.3.vsix` 文件。（或者在 Releases 页面下载，如果有）
2.  **打开 VS Code**。
3.  **进入扩展视图**：点击左侧边栏的方块图标，或按快捷键 `Ctrl+Shift+X`。
4.  **从 VSIX 安装**：点击扩展视图右上角的三个点 `...` 菜单，选择 **Install from VSIX... (从 VSIX 安装...)**。
5.  **选择文件**：在弹出的窗口中选择刚才下载的 `.vsix` 文件。
6.  安装完成后，Antigravity 的命令自动执行功能应该就能正常工作了。
